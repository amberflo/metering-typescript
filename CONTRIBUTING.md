# Contributors Guide

## Setting up the dev environment

Make sure you have a recent version of Node and run:
```
npm install
```

## Testing

Run all tests:
```
npm run test
```

## Linting & Checking

Lint the codebase:
```
npm run lint
```

Fix some lint errors automatically:
```
npm run fix
```

Check types:
```
npm run check
```

## Releasing

Build for release:
```
npm run build
```
